API reference for Angular Mosaic tabs

import {McTabsModule} from '@ptsecurity/mosaic/tabs';

Additional classes

McTab

Properties
NameDescription

ariaLabel: string

Aria label for the tab.

ariaLabelledby: string

Reference to the element that the tab is labelled by. Will be cleared if aria-label is set at the same time.

disabled: boolean

explicitContent: TemplateRef<any>

Template provided in the tab content that will be used if present, used to enable lazy-loading

implicitContent: TemplateRef<any>

Template inside the McTab view that contains an <ng-content>.

isActive: false

Whether the tab is currently active.

origin: number | null

The initial relatively index origin of the tab if it was created and selected after there was already a selected tab. Provides context of what position the tab should originate from.

position: number | null

The relatively indexed position where 0 represents the center, negative is left, and positive represents the right.

stateChanges: Subject<void>

Emits whenever the internal state of the tab changes.

templateLabel: McTabLabel

Content for the tab label given by <ng-template mc-tab-label>.

textLabel: string

Plain text label for the tab, used when there is no template label.
