API reference for Angular Mosaic style-manager

import {} from '@ptsecurity/false/style-manager';

Services

StyleManager

Class for managing stylesheets. Stylesheets are loaded into named slots so that they can be removed or changed later.

Methods
removeStyle

Remove the stylesheet with the specified key.

Parameters

key

string

setStyle

Set the stylesheet with the specified key.

Parameters

key

string

href

string
